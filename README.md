### ✍️ 진행 방식

- 주차 학습일 마다 학습 후 {number}-{이름}.md file에 정리하여 pr을 올린다.
- pr은 팀원들이 확인 후 코멘트와 approve를 통해서 검증된 뒤 merge한다.

- 매주 마지막에는 주차별 폴더 README.md 에 최종 정리하여 작성한다.
- 멘토을 Reviewer로 설정하고 PR을 올린다.
